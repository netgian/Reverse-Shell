Reverse Shell
=============

Simple reverse shell script for control the command prompt of a victim.


## Usage
On `server.py` put your `local ip` and your `open port`
```
HOST = "Local_IP"
PORT = 10000
```
And on `client.py` put your `public ip` and your `open port`
```
HOST = "Public_IP"
PORT = 10000
```

## Tested on:
- Windows 10
